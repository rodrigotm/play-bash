# play-bash
Help manage yours play servers on the bash terminal

Create alias playb

`cd /play-bash/`

After 

`alias playb='playb.sh'`

Commands:

`sudo playb start 9001 /opt/git/play-project/`

`sudo playb snapshot -s`

`sudo playb snapshot -r`

`sudo playb kill 9001`

`sudo playb kill -a`

<!-- `sudo restart 9001` -->
